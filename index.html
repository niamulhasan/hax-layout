Hax Layout

What id Hax Layout

Hax Layout is a super simple Responsive Layout System. It is really small in size the total size of this Layout about 3 kb.
Yes its 3 kb!!

It doesn't need any Javascript nor any libary like Jquery to work





Making Layout With Hax Layout

It is really easy to make a super responsive layout with Hax Layout. You can create Layouts with one sidebar, two sidebar, no sidebar or whatever you want.

You can use readymade layouts which are already included in this pack. Or you can make your own. Include hex.css and Use the CSS Style class shown bellow to make your layout. and Simply Don't use the class which div you don't need.


Code:

<div class="holder"> <div class="header">The Header</div> <div class="contener"> <div class="content sidebar"> Sidebar 1 </div> <div class="content main"> Main Content </div> <div class="content sidebar"> Sidebar 2 </div> </div> <div class="footer">The Footer</div> </div> If you want to style the divs, for example you want border of header or want to set background color of a div, then creat another .css file an create your own styles with the same class name.





Importent Note Using Hax Layout

Very Importent Thing Using this version of Hax Layout: Do Not use "px", "em" or anything like this as width in css style. Just use "%"
width: 250px; is Wrong. This is wrong.

sidebar { width: 250px; }


This is correct
sidebar { width: 25%; }
